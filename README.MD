# Proyecto API de Reservas de Hotel

Este proyecto es una API sencilla para gestionar reservas de hoteles.  

La idea del proyecto es poder **crear, ver, actualizar y eliminar reservas** de un hotel (CRUD).  
Cada reserva guarda información básica como el nombre del hotel, el tipo de habitación, la cantidad de huéspedes y las fechas de inicio y término.  

Al principio me costó un poco entender cómo funcionaban las rutas y cómo conectar los controladores, pero con el tiempo y con la ayuda de IA fui entendiendo mejor la lógica del servidor.

---

## Tecnologías utilizadas

- **Node.js** → para ejecutar JavaScript fuera del navegador.  
- **Express** → para crear el servidor y manejar las rutas.  
- **Swagger UI** → para documentar y probar los endpoints de forma visual.
